What i do

HTML

HTML (Hypertext Markup Language) is the standard language used to create web pages. It provides the structure and content of a webpage, using elements like headings, paragraphs, images, links, and forms. HTML uses a system of tags to define and organize the elements on a webpage, and it is a fundamental part of web development.

CSS

CSS (Cascading Style Sheets) is a style language used for describing the presentation and layout of web pages written in HTML. CSS allows web designers to control the colors, fonts, spacing, and positioning of elements on a webpage.

Javascript

JavaScript is a versatile and widely-used programming language for the web. It allows developers to add interactivity and dynamic behavior to websites. JavaScript can manipulate HTML and CSS, handle user input, make requests to servers, and create complex web applications

MYSQL

MySQL is an open-source relational database management system (RDBMS) that stores and manages structured data. It is widely used for building and managing databases in various applications, including websites and software systems. MySQL allows users to store, retrieve, and manipulate data efficiently using a structured query language (SQL).
